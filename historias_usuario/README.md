# Proyecto: LePetitDep - Gestión de Historias de Usuario

## Descripción breve
Este módulo contiene la especificación de requisitos y criterios de aceptación para la aplicación web ficticia **LePetitDep**, la cual permite a los usuarios ordenar café y productos de panadería para retirar en el local.

## Lista de Funcionalidades Principales
1. Registro de usuario
2. Inicio de sesión
3. Exploración del menú
4. Agregar productos al carrito
5. Realizar pedido
6. Historial de pedidos
